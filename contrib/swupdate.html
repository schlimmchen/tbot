tbot_contrib.swupdate¶

tbot_contrib.swupdate.swupdate_update_web(lh: Optional[tbot.machine.linux.lab.Lab], swu_file: tbot.machine.linux.path.Path, target_ip: str, timeout: int = 300) → None[source]¶

Upload an .swu file to a running swupdate server.

Parameters

  • lh (linux.Lab) – Optionally the lab-host from where to initiate the update.

  • swu_file (linux.Path) – Path to the .swu file (on the lab-host or locally).

  • target_ip (str) – IP-Address of the target host.

  • timeout (int) – Timeout.
